# Interview Task

Single-page React implementation for the provided fintech landing page reference.

## Stack

- React + Vite
- SCSS
- GSAP
- Framer Motion
- Poppins

## Features

- Component-based page architecture
- Reusable SCSS tokens, layout, and section styling
- Responsive navigation with mobile hamburger menu
- GSAP-driven load, scroll, and staggered section animations
- Framer Motion powered continuous `Industry Standard` marquee with center emphasis
- Lightweight interaction audio feedback
- Uses the supplied visual assets from `src/assets/images`
- Fluid responsive behavior across desktop, tablet, mobile, and small-mobile breakpoints

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Verification

- Production build verified with `npm run build`
